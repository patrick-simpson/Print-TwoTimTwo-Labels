# Privacy Policy - KVBC Kids Check-in Extension

The **KVBC Kids Check-in Extension** is designed with a "privacy-first" approach. 

### Data Collection
- **No Personal Data Collection:** This extension does not collect, store, or transmit any personal information to any external servers or third parties.
- **Local Processing Only:** All data extracted from the check-in page is processed locally within your browser and sent only to your own local machine via \http://localhost:3456\.
- **No Tracking:** We do not use cookies, tracking pixels, or any other analytics tools to monitor your behavior.

### Permissions
- **Storage:** Used only to save your local printer preferences (e.g., whether to use Auto-Print or the Print Dialog).
- **Host Permissions (localhost):** Required solely to communicate with the local KVBC Print Server running on your computer to facilitate silent printing.

### Contact
This extension is a utility tool for the KVBC Kids ministry. For any questions regarding this policy, please contact the project maintainer.
